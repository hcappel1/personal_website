A Hierarchical Multi-Robot Mapping Architecture Subject to Communication Constraints

Download PDF Github Repository Publication Link

Robots can generate maps of environments and locate themselves within these maps by using the simultaneous localization and mapping (SLAM) algorithm. The advantage of the slam algorithm is that it kills two bird with one stone by allowing robots to generate spacial maps and navigate these generated maps based on monte-carlo localization. SLAM is oftentimes performed by one robot to navigate an unknown space. However, teams of robots have the capability to significantly reduce the time needed to produce a complete map by merging individual maps into a global stitched map. This project is a part of my master's thesis and is based on an algorithm to coordinate teams of robots with a central computer to map an unknown space as efficiently as possible. The algorithm takes into account a major constraint oftentimes encountered by robot swarms. Each robot has a limited communication range. Therefore, the algorithm coordinates the robots making sure they stay within valid communication range of each other and the central computer.

Adaptive Compliance Shaping With Human Impedance Estimation

Download PDF Publication Link

Robotic exoskeletons are extremeley useful tools to allow a human to lift heavy objects, compensate for muscle loss, or perform difficult tasks. These devices can significantly augment the strength of the user, but have the potential to augment the strength to a greater degree and move more fluidly if controller stability guarantees are established. Because safety is always the number one priority, exoskeleton controllers oftentimes estimate human parameters within a large range to guarantee the stability of the controller for all users. However, human impedance estimation parameters can significantly improve the controller by reducing the conservative estimation of muscle stiffness and damping allowing for a more aggressive controller. We used an online muscle impedance estimator based on surface electromyograph sensors (sEMG) and stretch sensors placed around the arm. The estimator used a random forest regression to predict the impedance parameters in real time and fed this information to a compliance shaping controller within the exoskeleton. The paper was accepted by the American Control Conference 2020.

Finding Locomanipulation Plans Quickly in the Locomotion Constrained Manifold

Download PDF Github Repository Publication Link

Humanoid robots interact with their environment through locomotion (moving from one spot to another) and manipulation (using an end-effector to move an object). Oftentimes these processes are performed sequentially i.e. the robot will first move to a location and then manipulate the object as a separate task. However, robot efficiency and movement fluidity can be greatly improved if these two tasks are performed in parallel (in the same way we as humans do). This project generated locomanipulation trajectories for the NASA Valkyrie humanoid robot based on a neural network classifier. My contribution to this project involved developing the software for the footstep planner based on the a* graph search algorithm. The code is written in c++ and integrates into the ROS structure for the robot. The footstep planner produces a path of footsteps for the robot to perform and also plans a manipulation trajectory for the end effector to move. The paper was accepted by ICRA 2020.

An Optimal Control Strategy For a Multi-Task Whole Body Controller

Download PDF Github Repository

Whole body control allows robots with many joints to accomplish high level tasks without causing conflicts amongst the low level moving parts. For example a humanoid robot that wants to move its hand to a specific location needs the elbow shoulder and torso to coordinate their movements such that the hand gets to the correct location. While whole body control is an established field some improvements can be made to improve the efficiency by taking advantage of the hierarchy of tasks. This project aimed to develop an efficient optimal control strategy for achieving multiple high level tasks (moving the robot base and moving the end effector to a location) based on a sparse data structure from the system kinematics. This project was not published but remained a class project and a research proposal.
